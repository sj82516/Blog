19.Remove Nth Node From End of List 
題幹：給一個鍵結列，移除倒數n位的結 
解題：先算出列長，反推回來，找到length-n位移除

```Javascript
 */
/**
 * @param {ListNode} head
 * @param {number} n
 * @return {ListNode}
 */
var removeNthFromEnd = function(head, n) {
    var curr,next;
    var length = 0;
    curr = head;
    while(curr!==null){
        length++;
        curr = curr.next;
    }
    curr = head;
    var pos = length-n;
    if(pos===0){
        head = head.next;
        return head
    }
    while(pos>1){
        curr = curr.next;
        pos--;
    }
    curr.next = curr.next.next;
    return head;
};```
